Einführungsphase Frontend

Q3/2020 - Q4/2020

Erlernen der Grundtechniken Frontend

HTML/CSS/SASS/Javascript

HTML,CSS,JavaScript, die Grundpfeiler jeder Web-Entwicklung. Ohne sie, geht gar nichts. So kam auch ich nicht um diese Sprachen herum. Ebenso kam ich beim Thema CSS, auch an SASS/SCSS. Diese Sprachen beherrsche ich recht gut, aber sicherlich perfekt noch nicht. Jedoch lernte ich dabei, die Macht von Javascript oder genauer gesagt, dem ECMA-Script 6 kennen. Durch die Geschwindigkeit und der Vielseitigkeit, wollte und will ich noch immer, mehr über diese Sprache lernen, wie an meinem Stack auch zu erkennen ist.

UI/UX

UI und UX gehören meiner Meinung nach zusammen, wie auch das Bild zeigt, sind sie jedoch nicht eins. Mit dem ersten Schritt HTML/CSS/Javascript bin ich zwar in der Lage eine Webpage zu gestalten, aber wie sieht es mit der UserExperience aus? Genau diesem Thema werde ich mich Anfang Q4 2020 befassen, um eine entsprechend gute User Experience bieten zu können.

Begin BackEnd Main-Stack

Q2/2021

Abschluss des Backend-Parts des Main-Stacks

Node.JS

Eine Sache warum ich JavaScript so interessant finde, ist z.B. Node.JS. Durch Node.JS lässt sich Javascript auch außerhalb des Browsers ausführen, womit die Möglichkeiten unbegrenzt sind. Durch Node.JS ist Javascript nicht mehr "nur" die Sprache die einzelne Events im Browser verarbeitet, sondern auch zu I/O Prozessen genutzt werden kann, womit Javascript auch z.B. Zugriff auf das Filesystem ermöglicht wird. Bei Node.JS beherrsche ich die Basics.

Express.JS

Wer mit Node.js arbeitet, kennt sicherlich auch Express.JS. Ein Framework was die Arbeit mit Node.js ungemein erleichtert. Mir derzeit nur bekannt, im Rahmen eines Webservers. Von simplen Routing bis JSON Daten parsen sind mir die Vorgehensweisen bekannt, dennoch fehlt mir sicherlich ein wenig Übung.

MongoDB

Wer Daten strukturiert und sicher aufbewahren möchte, kommt um eine gute Datenbank nicht drum herum. Ebenfalls auf JavaScript aufbauend, befasste ich mich mit MongoDB. In Kombination mit mit Mongoose ein starkes Duo.
