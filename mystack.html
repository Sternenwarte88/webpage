Erlernen der Grundtechniken Frontend

HTML/CSS/SASS/Javascript (completed)

HTML, CSS, JavaScript, die Grundpfeiler jeder Web-Entwicklung. Ohne sie, geht gar nichts. So kam auch ich nicht um diese Sprachen herum. Ebenso kam ich beim Thema CSS, auch an SASS/SCSS. Diese Sprachen beherrsche ich recht gut, aber sicherlich perfekt noch nicht. Jedoch lernte ich dabei, die Macht von Javascript oder genauer gesagt, dem ECMA-Script 6 kennen. Durch die Geschwindigkeit und der Vielseitigkeit, wollte und will ich noch immer, mehr über diese Sprache lernen, wie an meinem Stack auch zu erkennen ist.

UI/UX (completed)

UI und UX gehören meiner Meinung nach zusammen, wie auch das Bild zeigt, sind sie jedoch nicht eins. Mit dem ersten Schritt HTML/ CSS/ Javascript bin ich zwar in der Lage eine Webpage zu gestalten, aber wie sieht es mit der User-Experience aus? Genau dieses Thema habe ich im November 2020 abgeschlossen, um eine entsprechend gute User Experience bieten zu können.

Abschluss des Frontendparts des Main-Stacks

React (completed)

React, ein voll individuell zusammenstellbares Framework. So multi-funktionell, wie auch einzigartig. Nicht nur im Bereich des Web-Developments gut einsetzbar, sondern auch im Bereich Web Progressiv Webapps. In Kombination mit Redux für kleine und große Vorhaben bestens geeignet. Als SPA oder Multipage oder auch nur teilweise in einer statischen HTML-Seite einsetzbar.

Redux (completed)

Redux ist ein sehr gutes Framework in Kombination mit React um verschiedene States einfacher zu handhaben. Es macht den Code um einiges leaner und besser verständlich.

Abschluss des Backend-Parts des Main-Stacks

Node.JS (completed)

Eine Sache warum ich JavaScript so interessant finde, ist z.B. Node.JS. Durch Node.JS lässt sich Javascript auch außerhalb des Browsers ausführen, womit die Möglichkeiten unbegrenzt sind. Durch Node.JS ist Javascript nicht mehr "nur" die Sprache die einzelne Events im Browser verarbeitet, sondern auch zu I/O Prozessen genutzt werden kann, womit Javascript auch z.B. Zugriff auf das Filesystem ermöglicht wird. Bei Node.JS beherrsche ich die Basics.

Express.JS (completed)

Wer mit Node.js arbeitet, kennt sicherlich auch Express.JS. Ein Framework was die Arbeit mit Node.js ungemein erleichtert. Mir derzeit nur bekannt, im Rahmen eines Webservers. Von simplen Routing bis JSON Daten parsen sind mir die Vorgehensweisen bekannt.

MongoDB (completed)

Wer Daten strukturiert und sicher aufbewahren möchte, kommt um eine gute Datenbank nicht drum herum. Ebenfalls auf JavaScript aufbauend, befasste ich mich mit MongoDB. In Kombination mit mit Mongoose ein starkes Duo.
